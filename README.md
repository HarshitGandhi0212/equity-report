## Methodology

Each report computes a **weighted composite risk score (0–100)** across three axes:

| Axis | Weight | What it measures |
|---|---|---|
| **Valuation** | 35% | P/E, P/B, EV/EBITDA, PEG, dividend yield, Mcap/Sales vs sector medians |
| **Financial Health** | 35% | Leverage, interest coverage, cash conversion, ownership stability, capital efficiency |
| **Growth** | 30% | Revenue, EBITDA, margin trajectory, order book and pipeline visibility |

- **Above 60** = elevated risk; caution before fresh entry
- **40–60** = moderate
- **Below 40** = balanced risk-reward

The score is a starting framework, not a verdict. Every report layers forensic and qualitative judgement on top.
